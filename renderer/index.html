Current Configuration

1 · Player Pool

Load the players leaving your CFB dynasty via college save file, or a CSV export.

Your CFB save MUST be Offseason Week 1 (the "Draft Stage"). This is the one point in a dynasty where the game has finalized exactly who's declaring for the draft. It cannot be any earlier or later week — other weeks either don't have this data yet or have already moved past it, and will give wrong or incomplete results.

No direct save access? Export your roster to a CSV instead (must include at minimum FirstName, LastName, Position, OverallRating columns) and load that.

No pool loaded

2 · Generate Class

Applies every setting from Position Weights, Physical Attributes, and Advanced to the loaded pool.

Recommended: generate once, then move on. Each run adds a little randomness on top of your settings (that's intentional — real draft classes have that same natural variance), so re-running produces a slightly different class every time. Only regenerate repeatedly if you're specifically testing how a Position Weight or Physical Attribute change affects the output on the Draft Class page — otherwise, generate once and write it into your franchise.
Nothing generated yet

3 · Write to Madden Franchise

Overwrites the incoming rookie class in your Madden 26 franchise save with your generated class.

Your Madden save CANNOT be in Preseason. The incoming rookie draft class doesn't exist yet during preseason, so there's nothing there for this tool to overwrite. Use a save from Regular Season, Postseason, or a later Offseason stage — anything except Preseason.

Class Construction

        

      


      

        
Bell-Curve Squeeze skill ratings

        
Skill ratings (Awareness, coverage, blocking, route running, etc.) get an extra cut concentrated at one tier of the class, fading out toward elites and deep bench players.

        

      


      

        
Development Traits

        
Controls how Normal, Star, Superstar, and X-Factor dev traits get handed out across the class.

        

      


      

        
Draft Projection draft order only

        
Where a player lands in the class (round/pick) can come from more than just overall -- CFB's own projected round, career awards, elite measurables, and position value all factor in. None of this touches a player's ratings.
