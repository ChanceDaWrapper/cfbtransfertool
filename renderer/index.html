Current Configuration

1 · Player Pool

Load the players leaving your CFB dynasty from your college save file.

Best accuracy at Offseason Week 1 (the "Draft Stage"). That's the one point in a dynasty where the game has finalized exactly who's declaring for the draft. Earlier saves work too -- the tool predicts who's likely to declare from Junior/Senior rosters -- but the closer you are to Week 1, the more accurate the class.

Loading a class from a CSV export instead of a save file is coming in a later version.

Auto-detect uses the game's official declared-draft-class list if your save has reached that point, and falls back to predicting declarations from Junior/Senior rosters otherwise -- so this also works on saves earlier than Offseason Week 1.

2 · Generate Class

Applies every setting from Position Weights, Rating Translation, Rating Categories, and Advanced to the loaded pool.

Recommended: generate once, then move on. Each run adds a little randomness on top of your settings (that's intentional — real draft classes have that same natural variance), so re-running produces a slightly different class every time. Only regenerate repeatedly if you're specifically testing how a Position Weight or rating change affects the output on the Draft Class page — otherwise, generate once and write it into your franchise.

3 · Export Draft Class File recommended

Builds a Madden Import Draft Class file from your generated class — no franchise save needed. In Madden: Franchise → Manage Roster → Import Draft Class and pick this file. It replaces the whole rookie class (no duplicates), with correct faces, positions, sizes, ratings, archetypes, and dev traits.

Madden's draft-class files are a fixed 402 players. Your generated class must have at least 402 (the top 402 by projection are used) — if it's smaller, raise Class Size on the Generate step and regenerate. Name the file whatever you like when saving — the app automatically adds the CAREERDRAFT- prefix Madden's importer requires, so you don't need to type it yourself.

4 · Write to Madden Franchise alternative

Overwrites the incoming rookie class in your Madden 26 franchise save with your generated class, writing directly into the save file (no in-game import step). The draft-class file above is usually cleaner.

Your Madden save CANNOT be in Preseason. The incoming rookie draft class doesn't exist yet during preseason, so there's nothing there for this tool to overwrite. Use a save from Regular Season, Postseason, or a later Offseason stage — anything except Preseason.

Rating Translation

        

          The heart of the converter: how a college rating becomes a Madden
          rookie rating. Physical traits barely change (a burner stays a
          burner), skills compress moderately, and mental ratings drop the most
          — because rookies rarely process at NFL speed. Every dial here is
          yours. Hover any field for details.

Per-Position Controls

        

          How hard each position compresses. Technical covers
          route running, coverage, blocking, pass rush, catching, and throw

Advanced

        
Class size, development traits, and draft-order projection weighting. Subtler effects than Position Weights or Rating Translation — each knob explains itself below.

        

          
        

      


      

        
Class Construction

        

      


      

        
Rating Realism on by default

        
Post-translation corrections, on by default. Turn one off if you're already handling the same thing with an external tool, so it isn't applied twice.

        

      


      

        
Development Traits

        
Controls how Normal, Star, Superstar, and X-Factor dev traits get handed out across the class.

        

      


      

        
Board Organization draft order only

        
Picks how the class gets ordered into rounds. CFB Projected Rounds follows the round CFB 27 projected for each player. Realistic Draft Day ignores that entirely, re-ranks the same players on talent, then lets some slide — giving you late-round steals instead of a clean best-to-worst sort. Either way the same players make the class; only where they get drafted changes.

        

      


      

        
Draft Projection draft order only

        
Where a player lands in the class (round/pick) can come from more than just overall -- CFB's own projected round, career awards, elite measurables, and position value all factor in. None of this touches a player's ratings.
